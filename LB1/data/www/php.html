PHP

Was ist PHP?

PHP (rekursives Akronym für PHP: Hypertext Preprocessor) ist eine weit verbreitete und für den allgemeinen Gebrauch bestimmte Open Source-Skriptsprache, welche speziell für die Webprogrammierung geeignet ist und in HTML eingebettet werden kann.

Nett, aber was heißt das genau? Ein Beispiel:

<!DOCTYPE html>
<html>
    <body>
                                
        <?php
            echo "Hallo, ich bin ein PHP-Skript!";
        ?>
                                
    </body>
</html>

Anstatt ein Programm mit vielen Anweisungen zur Ausgabe von HTML zu schreiben, schreibt man etwas HTML und bettet einige Anweisungen ein, die irgendetwas tun (wie hier "Hallo, ich bin ein PHP-Skript!" auszugeben). Der PHP-Code steht zwischen speziellen Anfangs- und Abschluss-Verarbeitungsinstruktionen <?php und ?>, mit denen man in den "PHP-Modus" und zurück wechseln kann.

PHP unterscheidet sich von clientseitigen Sprachen wie Javascript dadurch, dass der Code auf dem Server ausgeführt wird und dort HTML-Ausgaben generiert, die an den Client gesendet werden. Der Client erhält also nur das Ergebnis der Skriptausführung, ohne dass es möglich ist herauszufinden, wie der eigentliche Code aussieht. Sie können Ihren Webserver auch anweisen, alle Ihre HTML-Dateien mit PHP zu parsen, denn dann gibt es wirklich nichts, das dem Benutzer sagt, was Sie in petto haben.

Das Beste an der Verwendung von PHP ist, dass es für Neueinsteiger extrem einfach ist, aber auch einen riesigen Funktionsumfang für den professionellen Programmierer bietet. Scheuen Sie sich nicht, die lange Liste der PHP-Funktionen zu lesen. Sie können einsteigen, und binnen weniger Stunden bereits mit dem Schreiben von einfachen Skripten beginnen.

PHP Case Sensitivity

In PHP sind KEINE Schlüsselwörter (z.B. if, else, while, echo, etc.), Klassen, Funktionen und benutzerdefinierte Funktionen case-sensitive.

Im folgenden Beispiel sind alle drei folgenden echo-Aussagen gleich und gültig.

<!DOCTYPE html>
<html>
    <body>
                                                                
        <?php
            ECHO "Hello World!<br>";
            echo "Hello World!<br>";
            EcHo "Hello World!<br>";
        ?>
                                                                
    </body>
</html>

Schauen Sie sich das folgende Beispiel an; nur die erste Anweisung zeigt den Wert der Variablen $color an! Das liegt daran, dass $color, $COLOR und $coLOR als drei verschiedene Variablen behandelt werden.

<!DOCTYPE html>
<html>
    <body>
                                                                                                    
        <?php
            echo "My car is " . $olor . <br>";
            echo "My car is " . $COLOR . <br>";
            echo "My boat is " . $coLOR . <br>";
        ?>
                                                                                                    
    </body>
</html>

Kommentare in PHP

Ein Kommentar im PHP-Code ist eine Zeile, die nicht als Teil des Programms ausgeführt wird. Sein einziger Zweck ist es, von jemandem gelesen zu werden, der sich den Code ansieht.

Kommentare können dazu verwendet werden:

  • Lassen Sie andere Ihren Code verstehen
  • Erinnern Sie sich an das, was Sie getan haben - Die meisten Programmierer haben die Erfahrung gemacht, dass sie nach ein oder zwei Jahren wieder an ihre eigene Arbeit zurückkehren und neu herausfinden müssen, was sie getan haben. Kommentare können Sie daran erinnern, was Sie dachten, als Sie den Code geschrieben haben.

PHP unterstützt mehrere Arten von Kommentaren:

<!DOCTYPE html>
<html>
    <body>
                                                                                                                            
        <?php
            // Dies ist ein einzeiliger Kommentar
            # Dies ist auch ein einzeiliger Kommentar
            /*
            Dies ist ein mehrzeiliger Kommentar-Block,
            der sich über mehrere 
            Zeilen erstreckt
            */
            // Sie können auch Kommentare verwenden, um Teile einer Codezeile auszulassen
            $x = 5 /* + 15 */ + 5;
        ?>
    </body>
</html>

PHP-Variablen
Variablen sind " Behälter " zur Speicherung von Informationen.

Erstellen (Deklarieren) von PHP-Variablen

In PHP beginnt eine Variable mit dem $-Zeichen, gefolgt von dem Namen der Variable:

<?php
$txt = "Hello world!";
$x = 5;
$y = 10.5;
?>

Nach der Ausführung der obigen Anweisungen wird die Variable $txt den Wert Hello world!, die Variable $x den Wert 5 und die Variable $y den Wert 10.5 enthalten.

Hinweis: Wenn Sie einer Variablen einen Textwert zuweisen, setzen Sie den Wert in Anführungszeichen um.

Hinweis: Im Gegensatz zu anderen Programmiersprachen hat PHP keinen Befehl zur Deklaration einer Variablen. Sie wird in dem Moment erstellt, in dem Sie ihr zum ersten Mal einen Wert zuweisen.

PHP-Variablen

Eine Variable kann einen kurzen Namen (wie x und y) oder einen aussagekräftigeren Namen (Alter, Autoname, Total_Volumen) haben.

Regeln für PHP-Variablen

  • Eine Variable beginnt mit dem $-Zeichen, gefolgt von dem Namen der Variable
  • Ein Variablenname muss mit einem Buchstaben oder dem Unterstrich-Zeichen beginnen
  • in Variablenname kann nicht mit einer Zahl beginnen
  • Ein Variablenname darf nur alphanumerische Zeichen und Unterstriche (A-z, 0-9 und _ ) enthalten.
  • Bei Variablennamen wird zwischen Groß- und Kleinschreibung unterschieden ($alter und $ALTER sind zwei verschiedene Variablen)

Ausgabe-Variablen

Die PHP echo-Anweisung wird oft verwendet, um Daten auf den Bildschirm auszugeben.

Das folgende Beispiel zeigt, wie man Text und eine Variable ausgibt:

<?php
$txt = "Hello world!";
echo "This is my variable $txt."
?>

Das folgende Beispiel erzeugt die gleiche Ausgabe wie im obigen Beispiel:

<?php
$txt = "Hello world!";
echo "This is my variable " . $txt . "."
?>

Im folgenden Beispiel wird die Summe zweier Variablen ausgegeben:

<?php
$x = 5;
$y = 4;
echo $x + $y;
?>

Hinweis: Sie werden im nächsten Kapitel mehr über die echo-Anweisung und die Ausgabe von Daten auf dem Bildschirm erfahren.

PHP ist eine locker geschriebene Sprache

Beachten Sie im obigen Beispiel, dass wir PHP nicht mitteilen mussten, welcher Datentyp die Variable ist.

PHP assoziiert automatisch einen Datentyp mit der Variable, abhängig von deren Wert. Da die Datentypen nicht im engeren Sinne gesetzt sind, können Sie Dinge wie das Hinzufügen eines Strings zu einem Integer tun, ohne einen Fehler zu verursachen.

In PHP 7 wurden Typdeklarationen hinzugefügt. Dies gibt eine Option, um den Datentyp anzugeben, der bei der Deklaration einer Funktion erwartet wird, und durch die Aktivierung der strikten Anforderung wird ein "Fataler Fehler" bei einer Typabweichung ausgelöst.

Sie werden mehr über strenge und nicht strenge Anforderungen und Datentyp-Deklarationen im Kapitel PHP-Funktionen erfahren.

PHP-Variablen Umfang

In PHP können Variablen an beliebiger Stelle im Skript deklariert werden.

Der Scope einer Variable ist der Teil des Skripts, in dem die Variable referenziert/verwendet werden kann.

PHP hat drei verschiedene Variablenbereiche:

  • local
  • global
  • static

Globaler und lokaler Geltungsbereich

Eine außerhalb einer Funktion deklarierte Variable hat einen GLOBAL SCOPE und kann nur außerhalb einer Funktion angesprochen werden

<?php
$x = 5; // global scope

function myTest() {
    // Die Verwendung von x innerhalb dieser Funktion führt zu einem Fehler
    echo "<p>Variable x inside function is: $x</p>";
}
myTest();

echo "<p>Variable x outside function is: $x</p>";
?>

Eine innerhalb einer Funktion deklarierte Variable hat einen LOCAL SCOPE und kann nur innerhalb dieser Funktion angesprochen werden:

<?php
function myTest() {
    $x = 5; // global scope
    echo "<p>Variable x inside function is: $x</p>";
}
myTest();

// Die Verwendung von x innerhalb dieser Funktion führt zu einem Fehler                        
echo "<p>Variable x outside function is: $x</p>";
?>

PHP Das globale Schlüsselwort

Das globale Schlüsselwort wird verwendet, um auf eine globale Variable innerhalb einer Funktion zuzugreifen.

Verwenden Sie dazu das globale Schlüsselwort vor den Variablen (innerhalb der Funktion): 

<?php
$x = 5;
$y = 10;

function mytest() {
    global $x, $y;
    $y = $x + $y;
}

myTest();
echo $y; // outputs 15                    
?>

PHP speichert auch alle globalen Variablen in einem Array namens $GLOBALS[index]. Der Index enthält den Namen der Variablen. Auf dieses Array kann auch aus Funktionen heraus zugegriffen werden und es kann verwendet werden, um globale Variablen direkt zu aktualisieren.

Das obige Beispiel kann wie folgt umgeschrieben werden:

<?php
$x = 5;
$y = 10;
                                
function mytest() {
    $GLOBALS['y'] = $GLOBALS['x'] + $GLOBALS['y']; 
}
                                
myTest();
echo $y; // outputs 15                    
?>

PHP Das statische Schlüsselwort

Normalerweise werden beim Beenden/Ausführen einer Funktion alle ihre Variablen gelöscht. Manchmal wollen wir jedoch, dass eine lokale Variable NICHT gelöscht wird. Wir brauchen sie für einen weiteren Auftrag.

Verwenden Sie dazu das statische Schlüsselwort, wenn Sie die Variable zum ersten Mal deklarieren:

<?php
function mytest() {
    static $x = 0;
    echo $x;
    $x++;
}

myTest();
myTest();
myTest();
?>

Dann, jedes Mal wenn die Funktion aufgerufen wird, hat diese Variable immer noch die Informationen, die sie seit dem letzten Aufruf der Funktion enthält.

Hinweis: Die Variable ist immer noch lokal für die Funktion.

Das PHP Echo Statement

Die echo Anweisung kann mit oder ohne Klammern verwendet werden: echo oder echo().

Das folgende Beispiel zeigt, wie man Text mit dem echo-Befehl ausgibt (beachten Sie, dass der Text HTML-Markup enthalten kann):

<?php
    echo "<h2>PHP is Fun!<h2>";
    echo "Hello World!<br>";
    echo "I'm about to learn PHP!<br>";
    echo "This ", "string ", "was ", "made ", "with multiple parameters.";
?>

Das folgende Beispiel zeigt die Ausgabe von Text und Variablen mit der echo-Anweisung: 

<?php
$txt1 = "Learn PHP";
$txt2 = "luis-luescher.com";
$x = 5;
$y = 4;

echo "<h2>" . $txt1 . "<h2>";
echo "Study PHP at " . $txt2 ."<br>";
echo $x + $y;
?>

Die PHP Print-Anweisung

Die print Anweisung kann mit oder ohne Klammern verwendet werden: print oder print().

Das folgende Beispiel zeigt, wie man Text mit dem Print-Befehl ausgibt (beachten Sie, dass der Text HTML-Markup enthalten kann):

<?php
    print "<h2>PHP is Fun!<h2>";
    print "Hello World!<br>";
    print "I'm about to learn PHP!<br>";
?>

Das folgende Beispiel zeigt, wie Sie mit der Print-Anweisung Text und Variablen ausgeben können:

<?php
$txt1 = "Learn PHP";
$txt2 = "luis-luescher.com";
$x = 5;
$y = 4;
                            
print "<h2>" . $txt1 . "<h2>";
print "Study PHP at " . $txt2 ."<br>";
print $x + $y;
?>

PHP-Datentypen

Variablen können Daten unterschiedlichen Typs speichern, und unterschiedliche Datentypen können unterschiedliche Dinge tun.

PHP unterstützt die folgenden Datentypen:

  • String
  • Integer
  • Float (floating point numbers - also called double)
  • Boolean
  • Array
  • Object
  • NULL
  • Resource

PHP String

Ein String ist eine Folge von Zeichen, wie "Hallo Welt!".

Ein String kann ein beliebiger Text in Anführungszeichen sein. Sie können einfache oder doppelte Anführungszeichen verwenden:

<?php
$x = 'Hello world!';
$y = 'Hello world!';
                                                            
echo $x;
echo "<br>";
echo $y;
?>

PHP Integer

Ein ganzzahliger Datentyp ist eine nicht-dezimale Zahl zwischen -2.147.483.648 und 2.147.483.647.

Regeln für Integer:

  • Ein Integer muss mindestens eine Stelle haben.
  • Ein Integer darf keinen Dezimalpunkt haben.
  • Ein Integer kann entweder positiv oder negativ sein.
  • Integer können angegeben werden in: dezimaler (zur Basis 10), hexadezimaler (zur Basis 16), oktaler (zur Basis 8) oder binärer (zur Basis 2) Schreibweise

Im folgenden Beispiel ist $x ein Integer. Die PHP-Funktion var_dump() gibt den Datentyp und den Wert zurück:

<?php
$x = 5985;                                                                                      
var_dump($x);
?>

PHP Float

Ein Float (Fließkommazahl) ist eine Zahl mit einem Dezimalpunkt oder eine Zahl in exponentieller Form.

Im folgenden Beispiel ist $x ein Float. Die PHP-Funktion var_dump() gibt den Datentyp und den Wert zurück:

<?php
$x = 10.365;                                                                                      
var_dump($x);
?>

PHP Boolean

Ein Boolean repräsentiert zwei mögliche Zustände: WAHR oder FALSCH.

Booleans werden oft in konditionalen Tests verwendet.

$x = true;
$y = false;

PHP Array

Ein Array speichert mehrere Werte in einer einzigen Variablen.

Im folgenden Beispiel ist $cars ein Array. Die PHP-Funktion var_dump() gibt den Datentyp und den Wert zurück:

<?php
$cars = array("Volvo","BMW","Toyota");
var_dump($cars);
?>

PHP Object

Ein Objekt ist ein Datentyp, der Daten und Informationen über die Verarbeitung dieser Daten speichert.

In PHP muss ein Objekt explizit deklariert werden.

Zuerst müssen wir eine Klasse von Objekt deklarieren. Dazu verwenden wir das Schlüsselwort class. Eine Klasse ist eine Struktur, die Eigenschaften und Methoden enthalten kann:

<?php
class Car {
    function Car() {
            $this->model = "VW";
    }
}

// Ein Objekt erstellen
$herbie = new Car();

// Objekteigenschaften anzeigen
echo $herbie->model;
?>

PHP NULL Value

Null ist ein spezieller Datentyp, der nur einen Wert haben kann: NULL.

Eine Variable vom Datentyp NULL ist eine Variable, der kein Wert zugeordnet ist.

Tipp: Wenn eine Variable ohne Wert angelegt wird, wird ihr automatisch der Wert NULL zugewiesen.

Variablen können auch geleert werden, indem der Wert auf NULL gesetzt wird:

<?php
$x = "Hello world!";
$x = null;
var_dump($x);
?>

PHP-Ressource

Der spezielle Ressourcentyp ist kein eigentlicher Datentyp. Es ist die Speicherung einer Referenz auf Funktionen und Ressourcen außerhalb von PHP.

Ein gängiges Beispiel für die Verwendung des Ressourcen-Datentyps ist ein Datenbankaufruf.

Wir werden hier nicht über den Ressourcentyp sprechen, da es sich um ein fortgeschrittenes Thema handelt.
