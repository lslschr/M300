Funktionen

SQL - IF

Die IF-Funktion liefert entweder den Wert für TRUE oder FALSE zurück, je nach dem wie die angegebene Bedingung ausfällt.

SELECT CONCAT(IF(Punkte > 30000, Punkte,
'unterhalb der Grenze')) AS Punkte
FROM HighScores
ORDER BY Punkte DESC
;

SQL - SUM

Werte summieren

Durch die SUM() Funktion ist es möglich Werte einer Spalte aufsummieren zu lassen.

Integer aufsummieren
SELECT SUM(DateDiff(minute, 0, DURATION)) As
'TOTAL TIME' FROM Taks
GROUP BY ID
;
Zeiten aufsummieren
SELECT SUM(QUANTITY) As 'TOTAL' FROM Items
GROUP BY ID
;
