Linux Grundbefehle

In der untenstehenden Tabelle sind einige Grundbefehle für Linux aufgeführt.

Befehl Beschreibung
ls Auflistung des Verzeichnisse
ls -al Formatierte Auflistung mit versteckten Dateien
ls -lt Sortieren der formatierten Liste nach Zeitänderung
cd dir Auf dir Verzeichnis wechseln
cd Wechseln ins Hauptverzeichnis
pwd Aktuelles Arbeitsverzeichnis anzeigen
mkdir dir Erstellen eines dir Verzeichnisses
cat >file Setzt die Standardeingabe in die Datei
more file Den Inhalt der Datei ausgeben
head file Ausgabe der ersten 10 Zeilen der Datei
tail file Ausgabe der letzten 10 Zeilen der Datei
tail -f file Den Inhalt der Datei ausgeben, wenn sie wächst, beginnend mit die letzten 10 Zeilen
touch file Datei erstellen oder aktualisieren
rm file Löschen der Datei
rm -r dir Löschen des Verzeichnisses
rm -f file Erzwingen des Entfernens der Datei
rm -rf dir Erzwingen der Entfernung des Verzeichnisses dir
cp file1 file2 Kopieren des Inhalts von Datei1 nach Datei2
cp -r dir1 dir2 Kopiere dir1 nach dir2;erstelle dir2 wenn nicht vorhanden
mv file1 file2 Datei1 in Datei2 umbenennen oder verschieben, falls sich Datei2 in einem Verzeichnis befindet
ln -s file link Symbolischen Link zur Datei erstellen
ps So zeigen Sie die aktuellen Arbeitsprozesse an
top Alle laufenden Prozesse anzeigen
kill pid Beenden des Prozesses mit gegebener pid
killall proc Beenden der gesamten Prozesses namens proc
pkill pattern Tötet alle Prozesse, die dem Muster entsprechen
bg Gestoppte oder Hintergrund-Jobs auflisten, einen gestoppten Job im Hintergrund wieder aufnehmen
fg Bringt die jüngste Aufgabe in den Vordergrund
fg n Bringt den Job n in den Vordergrund
chmod file Ändern der berechtigung der Datei
grep pattern file Suche nach einem Muster in der Datei
grep -r pattern dir Rekursiv nach Muster in dir suchen
command | grep pattern Suchmuster in der Ausgabe eines Befehls
locate file Alle Instanzen der Datei finden
find . -name filename Sucht im aktuellen Verzeichnis und darunter, für Dateien und Verzeichnisse mit Namen, die mit dem Dateinamen beginnen
pgrep pattern Sucht nach allen genannten Prozessen, die stimmt mit dem Muster überein und gibt standardmäßig ihre ID
date Anzeige des aktuellen Datums und der Uhrzeit
cal Den Kalender dieses Monats anzeigen
uptime Aktuelle Betriebszeit anzeigen
w Anzeigen, wer online ist
whoami Als wen Sie eingeloggt sind
finger user Informationen über den Benutzer anzeigen
uname -a Kernel-Informationen anzeigen
cat /proc/cpuinfo CPU Informationen
cat proc/meminfo RAM Informationen
man command Das Benutzerhandbuch für Befehle anzeigen
df Anzeige der Festplattennutzung
du Speicherplatzausnutzung anzeigen
free Speicher- und Swap-Nutzung anzeigen
whereis app Mögliche Standorte der Applikation anzeigen
which app Anzeigen, welche Anwendungen standardmäßig ausgeführt werden
tar cf file.tar file Erstellen einer Datei mit dem Namen file.tar, die eine Datei enthält
tar xf file.tar Entpacken Sie der Dateien aus file.tar
tar czf file.tar.gz files Erstellen eines tar mit Gzip-Komprimierung
tar xzf file.tar.gz Tar mit Gzip extrahieren
tar cjf file.tar.bz2 Tar mit Bzip2-Komprimierung erstellen
tar xjf file.tar.bz2 Extrahieren eines Tar mit Bzip2
gzip file Komprimiert die Datei und benennt sie in file.gz um
gzip -d file.gz Dekomprimiert file.gz zurück in die Datei
ping host Ping-Host und Ausgabe der Ergebnisse
whois domain Whois-Informationen für Domains abrufen
dig domain DNS-Informationen für die Domäne abrufen
dig -x host Reverse-Lookup-Host
wget file Datei herunterladen
wget -c file Fortsetzen eines gestoppten Downloads
ctrl+c Hält den aktuellen Befehl an
ctrl+z Stoppt den aktuellen Befehl, Fortsetzung mit fg im Vordergrund oder bg im Hintergrund
ctrl+d Abmelden der aktuellen Sitzung, ähnlich wie beim Beenden
ctrl+w Löscht ein Wort in der aktuellen Zeile
ctrl+u Löscht die ganze Zeile
ctrl+r Eingeben um einen aktuellen Befehl aufzurufen
!! Wiederhohlt den vorherigen Befehl
exit Abmelden der aktuellen Sitzung
