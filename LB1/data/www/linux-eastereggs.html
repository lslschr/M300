Easter Eggs in Linux

Niemand kann behaupten, dass die Tausende von Entwicklern, die die auf einem typischen Linux-Rechner verfügbare Software erstellen, immer völlig ernst sind. Linux hat in der Vergangenheit einige lustige "Ostereier" in seine Software integriert. In diesem Artikel werden wir dir einige lustige Befehle und Optionen erklären, um deinen Tag zu erleichtern. Nicht alle von ihnen sind "Ostereier", aber wir denken, dass du sie trotzdem genießen wirst.

Emacs Games

Installieren wir zunächst emacs. Auf Ubunut/Debian wäre das so:

sudo apt-get install emacs

Welche Games gibt es?

Du kannst herausfinden, welche Spiele verfügbar sind, indem du dir dieses Verzeichnis ansiehst:

cd /usr/share/emacs/*/lisp/play
ls
5x5.elc       decipher.elc    gametree.elc   meese.elc      spook.elc
animate.elc   dissociate.elc  gomoku.elc     morse.elc      studly.elc
blackbox.elc  doctor.elc      handwrite.elc  mpuz.elc       tetris.elc
bruce.el      dunnet.elc      hanoi.elc      pong.elc       yow.elc
bubbles.elc   fortune.elc     landmark.elc   snake.elc      zone.elc
cookie1.elc   gamegrid.elc    life.elc       solitaire.elc

Ausführen

Um sie auszuführen, öffne emacs

emacs

Als nächstes drücke die Esc-Taste, gefolgt von x (zur Ausführung), und dann den Namen des Spiels ein, das Du starten möchtest:

Esc-x
pong

Um Emacs zu beenden, wenn Du fertig bist, drück auf Strg, gefolgt von x, und dann Strg und c:

Ctrl-x
Ctrl-c

Hollywood

Mission impossible? - gibt es im Terminal nicht. Wer schon immer mal wissen wollte, wie echte Hacks in Szene gesetzt werden, kann das gleichnamige Programm hollywood 🇬🇧 ausprobieren. Ab Ubuntu 15.04 ist es Bestandteil der offiziellen Paketquellen und kann einfach installiert werden:

sudo apt-get install hollywood

Ausführen des Programm:

hollywood

sl

Steam Locomotive ist ein kleines Gagprogramm, das eine Lokomotive anzeigt, sofern man sich beim Befehl ls vertippt hat.

Installation:

sudo apt-get install sl

Ausführen des programm:

sl
